## Elite Highlights (Going Beyond Basics)

* **NGINX Advanced Configuration (Lab 10):** Instead of a default setup, I configured a custom server block on port 8000, implementing a static content server and understanding the principles of **Reverse Proxying** and **Load Balancing**.
* **Manual Compilation & Source Installation (Lab 11):** Gained mastery over the `make` utility and `git` workflows by installing **Neofetch** directly from source code, bypassing standard package managers for better customization.
* **Security Testing & Brute-Force (Lab 12):** Built a custom **FastAPI** authentication server to simulate a real-world target and performed a dictionary attack using **Hydra** to evaluate credential strength.
