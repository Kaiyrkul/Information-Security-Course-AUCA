Lab 8: Text Editing with Vim

Objective:
To master basic and advanced operations in the Vim text editor, including modal editing and automation.

Tasks performed:
1. Basic Editing: Created "file.txt" and practiced switching between Normal and Insert modes.
2. Content Management: Practiced deleting lines with 'dd' and clearing the entire file with ':%d'.
3. Automation: Created a Bash script "vim_batch.sh" to demonstrate non-interactive file processing (adding line numbers) using Vim commands.
4. Commands mastered: :w (save), :q (quit), :set number (interface), i (insert), ESC (normal mode).

Conclusion:
Learned how to use Vim both as an interactive editor and as a tool for automated text processing in scripts.
